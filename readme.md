# Site Institucional: Viação Itapemirim

---

## Descrição do Projeto

Este projeto consiste em uma **Landing Page** (Página de Aterrissagem) com foco em um site institucional, utilizando o tema e a identidade visual da **Viação Itapemirim** (ou Nova Itapemirim). O objetivo é aplicar e demonstrar conhecimentos de estruturação HTML e estilização CSS, incluindo:

* Design responsivo (mobile-first).
* Implementação de layouts modernos (como Flexbox).
* Estilização de elementos complexos (cards e formulários flutuantes).
* Criação de componentes essenciais (Header, Footer e Seções Informativas).
